    Auto Unsubscriber

    This program goes through every email in your inbox within a given time frame and opens every "unsubscribe" link that is found (which could be useful for eliminating large amounts of spam at once). Since my email account is with Outlook, I am using the IMAP server for Outlook; other email providers will have different servers (some other common ones are: Gmail: imap.gmail.com; Office365: outlook.office365.com; Yahoo: imap.yahoo.mail.com). Note that if you are using Gmail, you will need to change your security settings to allow apps to access IMAP.

    On a side note, if you decide to install IMAPClient and you are using Windows, you may wish to consider installing the older version (use pip install imapclient==0.13 in the Command Line) since there have been SSL issues in the new version (unless you know how to configure your SSL settings, which I don't).

    Sushi Go Round

    Sushi Go Round is an online game available on Miniclip which involves satisfying customers' orders quickly by clicking on the approriate ingredients and ordering new ones when necessary. You can try out the first couple of levels here. This program uses the PyAutoGUI module, which allows GUI automation, to play the game automatically. Essentially, whenever a new image of sushi appears in the speech bubble area, the program will identify the image, check if enough ingredients are available, and, if so, click on the appropriate ones (if not, it will order more). In the meantime, it will pick up plates.

    For best results, use Chrome and make sure it is full-screen (since these settings were used to write the program and it is pixel-specific).

    You will also need to download these image files and make sure they are in the same directory as your program: download

    The challenging part was keeping track of which customers had been served and which customers hadn't, because we don't want to keep on making orders until a customer's speech bubble disappears (although the frequency at which customers appear makes this strategy more effective than one might expect). As a safety measure, in this program, a customer's order will be re-made if they have been waiting for longer than 10 cycles (in one cycle, all types of sushi will be checked amongst the orders). The worst case is when a customer appears suddenly at the leftmost side and takes an order that was made for someone else.

    The program isn't perfect, of course. The next direction for this project will probably be to track each order and ensure that it reached its recipient.

    Quiz Generator

    This program generates 30 multiple choice quizzes with the questions and answers in random order (which would be very tedious to do manually). I'm taking a class in genetics at the moment, so that's what in the example, but the question:answer pairs can be edited however you like.
